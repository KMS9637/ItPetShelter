## [ 사용자 기능 ]
### 정보 조회
- 사용자는 사이트 내의 정보를 조회 및 필터링을 할 수 있습니다.
### 예약 신청
- 사용자는 보호소 방문 예약 및 봉사활동을 신청할 수 있습니다.

## [ 관리자 기능 ]
### 게시판 관리
- 관리자는 로그인 시 보호소 및 동물 정보를 삽입, 삭제 및 수정할 수 있습니다.
### 예약 관리
- 관리자는 사용자가 예약한 정보를 조회할 수 있습니다.
